# Bus Pass Management System

##  Project Overview
This is a simple Python project that manages student bus passes. It is a menu-driven program that performs operations like adding, viewing, searching, and deleting bus pass records.

##  Features
- Add new bus pass
- View all bus passes
- Search pass by ID
- Delete pass by ID
- User-friendly menu system

##  Technologies Used
- Python 3.x
- Lists & Dictionaries
- Functions
- While Loop
